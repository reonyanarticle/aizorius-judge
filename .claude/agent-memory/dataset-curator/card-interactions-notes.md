card_interactions 候補15問（`evaluation/dataset-candidates-cards.jsonl`）を CR 2026-06-19 (en) と
Scryfall で検証した際の対応。カテゴリ設計・除外方針は `evaluation/reviews/REVIEW-cards.md`。See [[cr-version]]。

## Scryfall 取得の落とし穴
- `/cards/named?fuzzy=` は **`Accept: application/json` ヘッダが無いと 400 Bad Request**（User-Agent だけでは不足）。
- 日本語名は `cards/search?q=!"<en>" lang:ja&include_multilingual=true` の `printed_name` で取得。
- カード名の版注意: **Brokkos, Apex of Forever = 永遠の頂点、ブロコス**（"ブロッコス"や"Tyrant of Tetsuko"別カードと混同注意）。

## 要人間確認として notes に残した帰属（内容は確立裁定）
- エムラクール「唱えたとき」誘発が本体より先に解決: 601.2i(呪文が唱えられた瞬間)＋603.3(スタック配置)からの導出。CRに直接一文なし。Scryfall裁定で明示確認済み。
