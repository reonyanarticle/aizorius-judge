CR 2026-06-19 で照合済みの basic_rules 関連の要点。

**Why:** 同じルールを再検証する手間を省き、番号の版ずれ・混同を防ぐため。
**How to apply:** basic_rules / stack_priority / 状況起因処理の問を起案・検証するとき参照する。

- 敗北の状況起因処理: `704.5a`(ライフ0以下) / `704.5b`(空ライブラリからのドロー試行) / `704.5c`(毒10) / `704.5f`(タフネス0以下→墓地、再生不可) / `704.5g`(致死ダメージ→破壊、再生可) / `704.5i`(PW忠誠0) / `704.5j`(レジェンド・ルール)。全て `704.3`（優先権が発生するたびチェック）が土台。
- 704.5f(タフネス0)と704.5g(致死ダメージ)は別処理。前者は「破壊」でなく再生不可、後者は破壊で再生可。混同注意。
- ライブラリ切れ: 空になった瞬間では負けない。`704.5b`（引こうとした）＋ `104.3c`（残りを引いてから次に優先権が発生するとき敗北）。
- ロンドン・マリガンは `103.5` の1条にまとまっている（毎回7枚引き、キープ時にマリガン回数と同数を底へ）。
- 先手の初回ドロー・スキップは `103.8a`、明示的に **2人対戦限定**。
- 土地のプレイ: `305.1`(特別な処理・対応不可) `305.2`(通常1枚) `305.3`(自分のターン以外は不可) `116.2a`(特別な処理の定義) `505.6b`(メインでの条件)。
- 召喚酔い `302.6` は攻撃と{T}/{Q}コストの起動型能力のみ制限。それ以外の起動型能力は制限しない。
- ソーサリー・タイミング: `307.1`＋`505.6a`（メインのみ）。クリーチャー等は `306.1`/`505.6a`。
- 対象: 宣言は `601.2c`、変更不可の原則は `115.1`。解決時の立ち消え（全対象不適正で解決せず）は `608.2b`。
- 総コスト固定（locked in）は `601.2f`。マナ能力は支払い前 `601.2g`／部分払い不可 `601.2h`。
- マナ・プールは各ステップ・フェイズ終わりに空 `106.4`（ターン起因処理の本体は `500.5`）。
