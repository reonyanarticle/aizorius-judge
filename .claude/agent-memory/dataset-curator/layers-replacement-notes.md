CR 2026-06-19(en)で本文照合済みの、結論が一意に導ける裁定の番号対応（[[cr-version]]）。

## Layers (613)
- レイヤー順: 613.1a=L1コピー可能値, 613.1b=L2コントロール, 613.1c=L3テキスト, 613.1d=L4タイプ, 613.1e=L5色, 613.1f=L6能力, 613.1g=L7 P/T。
- コピー効果は**レイヤー1の副層1a**（613.2a）。L1適用後がコピー可能値（613.2c→707.2）。
- レイヤー7副層: 613.4a=7a(CDAのP/T), 613.4b=7b(設定/base参照), 613.4c=7c(修整・カウンター), 613.4d=7d(P/T入れ替え=最後)。
  - 「0/0になる→+1/+1カウンター=1/1」は 7b先→7c後（613.4b,613.4c）。
  - P/T入れ替えは常に最後（613.4d）。
- レイヤー2〜6内は**CDA先→他はタイムスタンプ順**（613.3）。
- 適用順の原則=タイムスタンプ（613.7）。**依存関係があれば優先**（613.8, 613.8a定義, 613.8b待機, 613.8c再評価）。

## Replacement (614/615/616)
- 複数の置換/軽減の適用順は**影響を受けるオブジェクトのコントローラー/オーナー or 影響を受けるプレイヤーが選ぶ**（616.1）。効果の発生源側ではない。同時なら101.4 APNAP。
- 特別な優先段階: 616.1a自己置換→616.1bコントロール変更→616.1cコピー化→616.1d裏向き→616.1e任意→616.1f繰り返し。
- 自己置換効果の定義=614.15（解決中の呪文/能力が自身の効果を置換）。
- 置換効果は自分自身を繰り返さない=**1イベントに1回だけ**（614.5）。置換された元イベントは発生せず修整後が発生（614.6）。
- 「〜が置かれた状態で戦場に出る/〜として出る」=出方変更の置換効果（614.1c,614.1d）。選択は**戦場に出る前**（614.12a）、判定は戦場に存在するならどうかで（614.12）。
- 再生=破壊置換効果（614.8）。「代わりに」はカード上に無いが定義に含意。「instead」語=置換効果(614.1a)。
- 0ダメージは与えない扱いで増幅置換は置換対象が無く無効（614.7a）— 未使用だが有効な問候補。

## 注意
- セクション番号だけ(例"613")は検証スクリプトがNG（個別番号必須）。
- 616.1系は「原則は影響を受ける側が選ぶ、ただし616.1a〜dの特別段階が例外」の構造。問の結論を原則/例外どちらに置くか明確に。
